"기록은 많은데, 필요할 때 못 찾는다."
지식 관리에서 가장 자주 터지는 문제다.

내 결론은 단순하다. **지식 저장소(Obsidian)와 실행 주체(OpenClaw)를 분리하되, 검색 경로는 단일화**해야 한다.
즉, 옵시디언은 지식의 원천(Source of Truth), OpenClaw는 그 지식을 꺼내 써서 실행하는 엔진으로 두는 구조다.

이 글은 `00. Inbox/Obsidian_RAG_Strategy.md` 초안을 기반으로, 실제 운영 가능한 블로그 버전으로 재정리한 내용이다.

## 1) 왜 "Obsidian + OpenClaw RAG" 조합이 실전에서 먹히는가

핵심은 세 가지다.

1. **원문 보존**: 노트는 계속 Obsidian의 마크다운으로 관리한다.
2. **의미 기반 검색**: OpenClaw의 `memory_search`가 키워드 불일치 상황에서도 관련 문맥을 찾는다.
3. **작업 즉시성**: 찾은 정보를 바로 실행(요약, 문서화, 후속 액션)으로 연결할 수 있다.

보통 실패하는 패턴은 "벡터 DB만 따로 키우고 원문 동기화가 깨지는" 경우다.
이 조합은 원문 중심이라 드리프트가 적고, 운영 난이도가 상대적으로 낮다.

## 2) 아키텍처 원칙: 저장과 추론을 분리하고, 검색만 연결한다

내가 추천하는 최소 원칙은 아래다.

- **저장 계층**: Obsidian Vault (`.md`) 
- **검색 계층**: OpenClaw `memory_search`
- **정제 계층**: 중요한 결정/요약만 `MEMORY.md` 혹은 프로젝트 노트로 환류

여기서 중요한 포인트 하나:

> OpenClaw 메모리 검색 설정은 **top-level `memorySearch`가 아니라**
> `agents.defaults.memorySearch` 아래에서 관리하는 것이 현재 문서 기준 정석이다.

(OpenClaw 공식 문서 `docs/concepts/memory.md` 기준)

## 3) 구현 순서 (실무형)

### Step A. 검색 범위 연결

Obsidian 전체를 한 번에 넣기보다, 우선 핵심 폴더부터 연결하는 게 안정적이다.

```json
{
  "agents": {
    "defaults": {
      "memorySearch": {
        "enabled": true,
        "extraPaths": [
          "/path/to/Obsidian/00. Inbox",
          "/path/to/Obsidian/10. Project",
          "/path/to/Obsidian/30. Resource"
        ]
      }
    }
  }
}
```

그리고 CLI에서 최소 확인:

```bash
openclaw memory status
```

### Step B. "정보 온도"로 인덱싱 우선순위 설계

모든 노트를 동등하게 취급하면 검색 품질이 금방 흐려진다.
그래서 나는 정보 온도로 나눈다.

- **Hot**: 최근 2주 프로젝트 메모, 의사결정 문서
- **Warm**: 실행 가능한 가이드, 반복 참고 문서
- **Cold**: 아카이브/아이디어 파편

운영 팁:
- Hot/Warm 영역을 `extraPaths` 우선 대상으로 둔다.
- Cold는 필요할 때만 확장한다.

### Step C. 대화 결과를 다시 지식으로 환류

RAG는 "찾기"로 끝나면 반쪽이다.
실전에서 가치가 나는 지점은 **결론이 다시 노트로 축적되는 순간**이다.

예:
- 대화에서 나온 핵심 의사결정 3줄 요약
- 실패 원인 + 재발 방지 체크리스트
- 다음 실행 액션(Owner/Deadline)

이 3가지를 프로젝트 노트로 남기면, 이후 검색 품질이 눈에 띄게 올라간다.

## 4) 품질 검증: RALF 루프(랄프 루프)로 운영하기

나는 아래 4단계를 한 세트로 돌린다.

### R — Relevance (관련성)
- 질문 대비 불필요한 문서가 섞이지 않는가?
- 답변 근거가 실제 프로젝트 문맥과 맞는가?

### A — Accuracy (정확성)
- 설정 키/경로/명령이 최신 문서와 일치하는가?
- 추정과 사실을 분리했는가?

### L — Logic (논리성)
- 결론까지 이어지는 추론 체인이 자연스러운가?
- 실행 순서가 실제 작업 흐름과 맞는가?

### F — Finish (완성도)
- 다음 사람이 바로 실행 가능한 체크리스트가 있는가?
- 문서만 읽고도 재현 가능한가?

이 루프를 통과하지 못한 글은 "지식"이 아니라 "메모"에 가깝다.

## 5) 운영 체크리스트 (바로 적용용)

- [ ] `agents.defaults.memorySearch` 기준으로 설정 정리
- [ ] Obsidian 연결 범위를 Hot/Warm부터 시작
- [ ] 주 1회: 중복 노트 병합
- [ ] 주 1회: 실패 사례 1건을 재발 방지 템플릿으로 변환
- [ ] 월 1회: 검색 미스 케이스 수집 후 태그/폴더 구조 보정

## 마무리

Obsidian + OpenClaw RAG의 본질은 "멋진 검색"이 아니다.
**결정과 실행의 흔적을 재사용 가능한 형태로 남기는 운영 시스템**이다.

기록이 자산이 되는지, 그냥 파일 더미가 되는지는
도구 선택보다 **운영 루프(RALF)**를 제대로 돌리느냐에 달려 있다.
